Si se puede crear el array (var milista = ['panadero', 456, [0, 1, 2]];). Es un array válido que tiene tres componentes: una cadena de texto ('panadero'), un número (456) y otro array ([0, 1, 2]).

Puede usar el código "var elemento = milista[2][2];"" para acceder a los elementos del array, en este caso accederá al 3º elemento (conjunto de números) y al 3º elemento de ese conjunto (en este caso el 2).
